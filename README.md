# Ecommerce Insights AI Agent

## Overview
Designed a semantic data layer and SQL-based analytics framework to generate actionable insights from ecommerce customer behavior data.

## Key Components
- semantic_view.sql → defines structured fact and dimension layer
- agent_sql.sql → supports query logic for analytics use cases

## Objective
To enable scalable and consistent analysis of customer behavior, revenue trends, and engagement metrics.

## Key Insights Enabled
- Revenue tracking and trend analysis
- Customer segmentation based on behavior
- User engagement analysis using session data

## Tools Used
- SQL (Snowflake-style modeling)
- Data analytics concepts

## Outcome
Designed a structured analytics layer that supports faster and more reliable business insights.
